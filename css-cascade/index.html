CSS 层叠与优先级演示

1. 相同属性的覆盖

这个元素的颜色由后面的属性决定 
.box-1 {
    color: blue;
    color: red;  /* 这个会生效 */
}

2. 选择器优先级

这个文本的样式由最高优先级的选择器决定 

p { color: blue; }                    /* 优先级: 0,0,0,1 */
.text { color: green; }               /* 优先级: 0,0,1,0 */
#special-text { color: red; }         /* 优先级: 0,1,0,0 */

3. !important 规则

这个元素的样式使用了 !important 
.box-2 {
    color: blue !important;  /* 这个会生效 */
    color: red;
}

4. 继承与覆盖

父元素文本
子元素文本 
.parent {
    color: blue;
}
.child {
    /* 继承父元素的颜色，但可以被覆盖 */
    color: red;
}

5. 复合属性与单独属性

复合属性示例 
.box-3 {
    margin: 20px;              /* 复合属性 */
    margin-top: 40px;          /* 覆盖单个值 */
    
    background: #f0f0f0;       /* 复合属性 */
    background-color: #e0e0e0; /* 覆盖特定值 */
}

6. 媒体查询中的覆盖

响应式布局示例（调整窗口大小查看效果） 
.responsive-box {
    width: 300px;
}
@media (max-width: 600px) {
    .responsive-box {
        width: 100%;  /* 在小屏幕上覆盖原有宽度 */
    }
}
